## Aritméticos
## Asignación

Operadores de Asignación

  • Asignación con Suma(+=): La expresión x += 2, es equivalente a la expresión x = x + 2.
  • Asignación con Resta(-=): La expresión x -= 2, es equivalente a la expresión x = x - 2.
  • Asignación con Multiplicación(*=): La expresión x *= 2, es equivalente a la expresión x = x * 2.
  • Asignación con División(/=): La expresión x /= 2, es equivalente a la expresión x = x / 2.
  • Asignación con Residuo(%=): La expresión x %= 2, es equivalente a la expresión x = x % 2.

Operadores de Asignación: Python

  • Asignación con Potenciación(**=): La expresión x **= 2, es equivalente a la expresión x = x ** 2.
  • Asignación con División exacta(//=): La expresión x //= 2, es equivalente a la expresión x = x // 2.
## Incremento
## Incremento El operador ++ se puede utilizar como un operador __prefijo__ (++i) o __postfijo__ (i++), y dependiendo de la posición donde se ubique entonces la asignación se hace antes o después del incremento.
## Preincremento __++x__: La variable x se incrementa en 1 y luego se retorna el nuevo valor de la variable.
## Preincremento: Ejemplo ```cpp #include <iostream> using namespace std; int main() { int x = 2; int y = ++x; cout << x << endl; cout << y << endl; return 0; } ```
## Preincremento: Ejemplo ```cpp #include <iostream> using namespace std; int main() { int x = 2; x += 1; int y = x; cout << x << endl; cout << y << endl; return 0; } ```
## Postincremento __x++__: Retorna el actual valor de la variable x y luego incrementa su valor en 1.
## Postincremento: Ejemplo ```cpp #include <iostream> using namespace std; int main() { int x = 2; int y = x++; cout << x << endl; cout << y << endl; return 0; } ```
## Postincremento: Ejemplo ```cpp #include <iostream> using namespace std; int main() { int x = 2; int y = x; x += 1; cout << x << endl; cout << y << endl; return 0; } ```
## Decremento
## Decremento El operador -- se puede utilizar como un operador __prefijo__ (--i) o __postfijo__ (i--), y dependiendo de la posición donde se ubique entonces la asignación se hace antes o después del incremento.
## Predecremento __--x__: La variable x se decrementa en 1 y luego se retorna el nuevo valor de la variable.
## Predecremento: Ejemplo ```cpp #include <iostream> using namespace std; int main() { int x = 2; int y = --x; cout << x << endl; cout << y << endl; return 0; } ```
## Predecremento: Ejemplo ```cpp #include <iostream> using namespace std; int main() { int x = 2; x -= 1; int y = x; cout << x << endl; cout << y << endl; return 0; } ```
## Postdecremento __x--__: Retorna el actual valor de la variable x y luego decrementa su valor en 1.
## Postdecremento: Ejemplo ```cpp #include <iostream> using namespace std; int main() { int x = 2; int y = x--; cout << x << endl; cout << y << endl; return 0; } ```
## Postdecremento: Ejemplo ```cpp #include <iostream> using namespace std; int main() { int x = 2; int y = x; x -= 1; cout << x << endl; cout << y << endl; return 0; } ```

## Relacionales

Operadores Relacionales

  • Mayor que: > (>).
  • Menor que: < (<).
  • Mayor o igual que: \(\geq\) (>=).
  • Menor o igual que: \(\leq\) (<=).
  • Igualdad: = (==).
  • Desigualdad: \(\neq\).
    C++ Python
    != !=
    <>
## Operadores Relacionales: Ejemplo ```cpp #include <iostream> using namespace std; int main() { int a = 0.5; bool f = (0 < a && a <= 1); cout << a << endl; cout << f << endl; return 0; } ```
## Operadores Relacionales: Ejemplo ```python a = 0.5 f = (0 < a and a <= 1) print(a) print(f) ```
